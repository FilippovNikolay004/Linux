= boot - Файлы загрузчика (загрузочные диски);
= etc (etcetera) - Файлы конфигурации (реестр);
= bit (binaries) - Бинарные файлы пользователя (исполняемые файлы);
= sbin (system binaries) - Системные бинарные файлы (исполняемые файлы);
= dev (devices) - Файлы устройств (все устройства представлены в виде файлов);
= lib/32/64 (library) - Системные библиотеки;
= mnt (mount) - Монтирование внешних или доп. файловые системы (т.е. нужно для
	монитрования (подключения) внешних файлов к ОС);
= media - Съемные носители, система монтирует все подключаемые внешние накопители (флешки, диски);
= root - Директория для суперпользователей "/";
= sys (system) - Информация о системе (получение информации от ядра. Позволяет изменять параметры работы системы);
= var (variable) - Переменные файлы (сис. файлы, БД, кэши);
= opt (optional app) - Дополнительные программы (игры, драйвера);
= run - процессы (после перезагрузки все файлы теряются);
= srv (server) - Содержат файлы серверов и сервисов;
= home - Домашняя директория "~";
= proc (proccess) - Информация о процессах (содержит всю информацию о запущенных процессах в реальном времени);
= usr (user app) - Программы пользователя (имеет большу структуру каталогов в Linux, такие как:
	исполняемые файлы, исходные программы, приложения, картинки, музыка, документы);
= tmp (temp) - Временные файлы системы, программ или пользователями. Файлы удаляются при каждой перезагрузке;
= cdrom - Монитрование для оптических дисков (CD или DVD);
= lost+found - Восстанавление для повреждённых или "потерянных" файлов;
= snap - используется для установки и управления программ в виде изолированых контейнеров;
